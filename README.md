# Frame Creatives Lab - Agency Website

A high-performance, cinematic web platform for **Frame Creatives Lab**, built with Next.js 14. This site showcases a subscription-based creative model for premium video editing and design services.

## 🚀 Tech Stack

* **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Deployment:** [Vercel](https://vercel.com/)
